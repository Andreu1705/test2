# Random Password Generator

**Автор:** Будников Артем
**Дата:** 18 мая 2026
## Описание
Графическое приложение на Python для генерации случайных паролей с настраиваемыми параметрами.  
Поддерживает сохранение истории паролей в JSON и её загрузку.

## Возможности
- Выбор длины пароля (4–64 символа)
- Выбор типов символов: цифры, буквы, спецсимволы
- Копирование пароля в буфер обмена
- История последних 20 паролей с датой и настройками
- Сохранение/загрузка истории из JSON
