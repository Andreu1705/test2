# Random Password Generator

**Автор:** Будников Артем
**Дата:** 18 мая 2026
## Описание программы
Random Password Generator — это GUI-приложение для генерации случайных паролей с возможностью настройки параметров, сохранением истории генерации в JSON-файл и загрузкой истории обратно. Приложение позволяет гибко настраивать состав символов в пароле и контролировать его длину.\

## Возможности
- ✅ Генерация случайных паролей с настраиваемой длиной (от 4 до 64 символов)
- ✅ Выбор типов символов: цифры, буквы (A-Z, a-z), спецсимволы (!@#$%^&*()-_=+[]{};:?/<>)
- ✅ Копирование сгенерированного пароля в буфер обмена
- ✅ История последних 20 сгенерированных паролей с датой, длиной и настройками
- ✅ Автоматическое сохранение истории в JSON-файл
- ✅ Автоматическая загрузка истории при запуске приложения
- ✅ Очистка всей истории с подтверждением
- ✅ Валидация вводимых данных (минимальная/максимальная длина, выбор хотя бы одного типа символов)
